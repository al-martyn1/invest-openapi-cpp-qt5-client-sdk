1) В API нет возможности узнать, когда произошел листинг инструмента на бирже

   Решение: приходится это делать поиском по свечам.


2) Свечи с высоким разрешением (1/2/3/5/10 и тп минут) появились гораздо позже, 
   чем свечи с разрешением в день/неделю/месяц.
   Возможно, что просто эти данные не хранятся слишком долго.

   Решение: тупо перебирать, и ждать, когда начнут приходить данные. Но это очень долго.


3) В API нет возможности узнать, какими инструментами оперировал пользователь 
   в течение своей брокерской жизни - есть только текущий список инструментов в портфеле.
   Запросы к портфелю возвращают только текущие ненулевые позиции.

   Решение: запрашивать все операции по каждому инструменту на бирже.
   Вариант: использовать список бумаг, который задаёт пользователь.
   Вариант: если будет софт, через который всё будет делаться централизовано, он
            должен запоминать использованные инструменты.
            Но для нового местоположния либо переносить конфиги, либо всё запрашивать с сервера.
